To tell git your identity (do this if you haven't used git on your computer before): 


git config --global user.name "Your Name"
git config --global user.email "youremail@gmail.com"

To tell git to take note of all the changes you've made: 

git add .

To tell git to "name" the added changes: 

git commit -m "Whatever message here"

To tell git to tell github about the committed changes: 

git push

To tell git to get any changes from github: 

git pull
