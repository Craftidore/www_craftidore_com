Problems for Spencer

Problem 1: Very Easy

Accept a number of arguments, then read in that many integers. Calculate the mean of those integers and print it out. The mean should be a double.

Example Input:

5
1 2 3 4 5

Example Output:

3.0

Example Input:

11
34 20 40 88 945 12 3 -6 96 92 92

Example Output:

128.727272

Problem 2: Easy

Accept a number of arguments, then read in that many space-separated words. Print them out sorted alphabetically, ignoring case. Do not use any built-in sort function or functionality of the language you're using. There are no speed requirements here, so using bubble sort is fine.

Example Input:

7
kitchen office television door satchel window lamp

Example Output:

door kitchen lamp office satchel television window

Example Input:

11
kiTChEn OFFICE Television doOr satchel WindoW lAMP

Example Output:

doOr kiTCHEn lAMP OFFICE satchel Television WindoW

Problem 3: Medium

Accept an x size and a y size (integers), then read in that many letters, ignoring whitespace. Start at 0,0 (top-left corner) and find the shortest number of cardinal moves (left-right-up-down, no diagonal) it would take to reach the bottom right corner. You may not move through the x characters and may not move outside the map.

Example Input:

6 5
......
......
......
......
......

Example Output:

9

Example Input:

6 5
.x....
.x.xx.
.x.x..
.x.x.x
...x..

Example Output:

19

Example Input:

10 10
...x......
.x.....x..
.xxxxx.xxx
.x.....x..
.x.x..xx.x
.x.x......
.xxxxxxxx.
.x...x....
.x.x.x.x..
...x...x..

Example Output:

20

Problem 4: Easy

Read in space-separated words until 'end' then read in words until 'end' again. Print out any words in the second batch that aren't present in the first batch.

Example Input:

apple banana cookie danish eggs end apple biscut cereal end

Example Output:

biscut cereal

Example Input:

Foust Adair Ragsdale Hunt Fields McCown end Foust Ragsdale Baird Steil Fields McCown end

Example Output:

Baird Steil
