Problems for Julie

Problem 1:

Accept a number of arguments, then read in that many integers. Calculate the mean of those integers and print it out. The mean should be a double.

Example Input:

5
1 2 3 4 5

Example Output:

3.0

Example Input:

11
34 20 40 88 945 12 3 -6 96 92 92

Example Output:

128.727272

Problem 2:

Accept a number of arguments, then read in that many integers. Find the largest odd number.

Example Input:

5
1 2 3 4 5

Example Output:

5

Example Input:

11
34 20 40 88 946 12 3 -6 96 92 92

Example Output:

3

Problem 3:

Accept a number of arguments, then read in that many words. Print them out in alphabetical order. Case should not matter.

Example Input:

5
ff ab aa ad ac ae

Example Output:

aa ab ac ad ae ff

Example Input:

11
Barry John Extremely iPhone Julie SuMMit 
Original width example summation cooKIES

Example Output:

Barry cooKIES example Extremely iPhone John Julie Original summation SuMMit width

Problem 4:

Read in words until 'end' then read in words until 'end' again. Print out any words in the second batch that aren't present in the first batch.

Example Input:

apple banana cookie danish eggs end
apple biscut cereal end

Example Output:

biscut cereal

Example Input:

Foust Adair Ragsdale Hunt Fields McCown end
Foust Ragsdale Baird Steil Fields McCown end

Example Output:

Baird Steil
