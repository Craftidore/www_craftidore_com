We Have Time Requirements

Presentation by Ty Bates and Elliot Hodge

Performance != Responsiveness

Performance:

  • Computation over time
  • Processing large amounts of input quickly

Responsiveness:

  • User perception of app speed
  • Acknowledging user input quickly
  • Keeping the user informed

Responsiveness matters more to user experience than performance.

Conclusion

Highly responsive applications put users in control by quickly acknowledging each user request, by providing continuous feedback about progress toward fulfilling each request, and by letting users complete tasks without unacceptable delays.
